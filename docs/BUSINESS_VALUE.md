## 一、我們在做什麼

**CK_Missive 是一套給乾坤測量工程（及類似專業服務公司）使用的
「公文 + 案件 + 財務 + AI 助理」整合平台。**

它解決四件事：

1. **公文管理不再靠 Excel + LINE 群組**
   — 收發文、截止日、批次操作、來文單位歷史一鍵可查
2. **派工進度不再靠人工彙整**
   — 晨報自動生成、逾期自動偵測、里程碑自動追蹤
3. **財務核銷不再靠紙本 + 個人試算表**
   — 報價/開票/請款/費用/資產一條龍，跨案件跨廠商跨委託單位視圖
4. **AI 助理替你回答「這件事到底怎樣了？」**
   — 用中文自然語言問，AI 從 1,698 份公文 + 82 個案件 + 127 筆派工裡即時找答案

## 二、v5.7 帶來的商業價值

### 「AI 助理有記憶了」

**之前**：每次問 AI 都得從頭講一遍背景。
**之後**：AI 記得上週你問過什麼、哪個專案你特別關注、你的表述習慣。

**商業翻譯**：
- **省 30% 溝通時間**：不必重複背景
- **個人化建議**：AI 主動提醒「上週問的 A 案今天到期了」
- **知識累積**：每週自動生成「本週進度自傳」，變成公司知識資產

**技術背後**：Memory Wiki 7-Phase（自傳 + 模式萃取 + 結晶 + 反迴聲）— 但用戶不需要懂。

---

## 三、v5.8 帶來的商業價值

### 「AI 助理有人格了」

**之前**：AI 回答四平八穩、像客服機器人。
**之後**：AI 用「坤哥」這個人格跟你對話 — 懂工程、敢質疑、會用繁中行話。

**商業翻譯**：
- **更順暢的 ad-hoc 討論**：像跟同事聊，不像跟系統講話
- **反迴聲質疑機制**：AI 會主動挑戰不合理的決策（避免「Yes-Man」陷阱）
- **身份統一**：你在 LINE / Telegram / Web UI 遇到的都是同一個「坤哥」

**倫理紅線**：
- 不編造公文內容
- 不替你簽名核章
- 不送出未經確認的正式文件
- 敏感內容自動遮罩（身分證 / 金額 / 長編號）

## 七、對「AI 助理有人格」的聲明

**坤哥意識體是一個「有設計的對話人格」，不是通用 AI。**

- 它只討論乾坤工程相關業務（公文、案件、派工、財務、標案）
- 它不假裝有情感、不假裝是人類
- 它可以被關閉、被替換、被訓練
- 它的「人格定義」寫在 `wiki/SOUL.md`（純文字，隨時可改）

這個設計目的：
- 讓你用同事的感覺使用系統，降低摩擦
- 讓系統有一致的風格，不會今天熱情明天冷淡
- 讓內部討論可以引用「坤哥怎麼說」，形成共同語言

這個設計**不是**：
- 它不會替代同事
- 它不會做最終決策
- 它不會承擔法律責任
