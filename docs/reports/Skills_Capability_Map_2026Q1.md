## 三、核心智能詳解

### 3.1 自我學習系統

```
用戶查詢 ──────────────────────────────────────────────────────────→ 回答
    │                                                                 │
    │  [背景非阻塞]                                                    │
    │                                                                 │
    ├─→ ① 模式學習 ──→ MD5模板 + 語意匹配 ──→ Redis (max500, 7d衰減)  │
    │                                                                 │
    ├─→ ② 自我評估 ──→ 5維度加權評分 ──→ 信號佇列 (severity分級)       │
    │       │          相關性 0.30                                     │
    │       │          完整度 0.25                                     │
    │       │          引用   0.20                                     │
    │       │          延遲   0.10                                     │
    │       │          效率   0.15                                     │
    │       │                                                         │
    ├─→ ③ 進化排程 ──→ 每50次/24h觸發 ──→ 提升/降級/清理模式         │
    │                                                                 │
    └─→ ④ 跨會話學習 ──→ DB持久化 ──→ cosine過濾 ──→ 注入LLM prompt   │
                                                                      │
    [下次查詢] ◄── 受益：升級模式+精準路由+歷史學習 ◄────────────────┘
```

| 參數 | 值 | 可配置 | 配置來源 |
|------|---|--------|---------|
| 模式上限 | 500 | agent-policy.yaml | pattern_learner.max_patterns |
| 衰減週期 | 7d | agent-policy.yaml | pattern_learner.decay_days |
| 語意閾值 | 0.85 | agent-policy.yaml | pattern_learner.semantic_threshold |
| 進化觸發 | 50次/24h | agent-policy.yaml | evolution.trigger_every_n_queries |
| 自評維度 | 5 | 程式碼 | agent_self_evaluator.py |
| 冷啟動種子 | 29 (7類) | 程式碼 | pattern_seeds.py |
| 上下文窗口 | 自適應 2/4/6 | agent-policy.yaml | adaptive_context_* |

### 3.2 語音辨識

```
┌──────────┐    ┌─────────────┐    ┌──────────────┐    ┌──────────┐
│ 音訊輸入  │───→│VoiceTranscr.│───→│ Agent Query  │───→│ 文字回答  │
│ LINE/API │    │             │    │ Orchestrator │    │          │
└──────────┘    │ ┌─────────┐ │    └──────────────┘    └──────────┘
                │ │Groq主路徑│ │
                │ │whisper   │ │    支援格式: m4a/mp3/wav/webm/ogg/flac
                │ │v3-turbo │ │    語言: 中文(CJK prompt)
                │ └────┬────┘ │    超時: Groq 60s / Ollama 120s
                │      │失敗  │    快取: Redis 24h TTL
                │ ┌────▼────┐ │    對話記憶: Redis session (line:{user_id})
                │ │Ollama備用│ │
                │ └─────────┘ │
                └─────────────┘
```

| 對話記憶 | 已實作 (Redis session) | 延緩啟用 (LINE) |
| 即時串流 | 未實作 | WebSocket 串流轉錄 |
| 多語言 | 中文優化 | 自動語言偵測 |
| 語音回覆 | 未實作 | TTS 整合 |

### 3.3 影像辨識 (OCR)

```
┌──────────┐    ┌─────────────────────────────────┐    ┌──────────┐
│ 圖檔/PDF │───→│ tool_executor_document           │───→│ 提取文字  │
│ 附件上傳  │    │                                  │    │          │
└──────────┘    │  ┌─ 圖片 (PNG/JPG/TIFF/BMP)     │    │ ┌──────┐ │
                │  │  ① EXIF 旋轉校正              │    │ │ NER  │ │
                │  │  ② Tesseract (chi_tra+eng)    │    │ ├──────┤ │
                │  │  ③ timeout=30s 保護           │    │ │ 摘要  │ │
                │  │                                │    │ ├──────┤ │
                │  ├─ PDF                           │    │ │Embed │ │
                │  │  ① PyPDF2 文字提取             │    │ └──────┘ │
                │  │  ② 文字<50字 → 渲染為圖片     │    └──────────┘
                │  │  ③ OCR回退 (每頁獨立)          │
                │  │  ④ 一頁失敗不影響全文          │
                │  └────────────────────────────────│
                └──────────────────────────────────-┘
```

| 掃描PDF | 自動OCR回退 | 品質分數回報 |
| 旋轉校正 | EXIF transpose | 傾斜校正 (deskew) |
| 超時保護 | 30s/頁 | 可配置 |
