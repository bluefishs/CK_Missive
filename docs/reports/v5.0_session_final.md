## 四、已知限制

| 限制 | 原因 | 影響 |
|------|------|------|
| 同 session 連續查詢降級 | 歷史 context 污染 LLM 規劃 | 中 |

| 統計查詢 LLM 合成不引用 summary | 合成 prompt 未強制引用 | 低 |
