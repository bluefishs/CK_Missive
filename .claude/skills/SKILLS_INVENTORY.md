## 版本管理規範

### 版本號格式
```
MAJOR.MINOR.PATCH

MAJOR: 重大架構變更
MINOR: 新增功能或重要更新
PATCH: Bug 修復或小幅改善
```

### 狀態定義

| 狀態 | 符號 | 說明 |
|------|------|------|
| 維護中 | ✅ | 積極維護，定期更新 |
| 穩定 | 🟢 | 功能完整，偶爾更新 |
| 過時 | ⚠️ | 需要審查或更新 |
| 廢棄 | ❌ | 計畫移除 |

### 更新程序

1. 修改 Skills 檔案
2. 更新檔案內的版本號
3. 更新本清單
4. 更新 `CLAUDE.md` (如需要)
5. 提交變更
